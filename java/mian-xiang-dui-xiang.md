## 类和对象

**类：** 是一种自定义的数据类型。有时把这种数据类型也叫做「类类型」或者「引用数据类型」。「引用」就是内存地址的意思。

**对象：**通过类创建的变量。

> 类是一群对象的特征母版，对象是类的具体实例。
>
> 类是一群对象的抽象。

### 类的定义

> 类所具备的最基本要素：（静态）属性、（动态）方法。

语法：

```java
[修饰符] class 类名 {
    // 成员变量
    [修饰符] 数据类型 成员变量1;
    [修饰符] 数据类型 成员变量2;
    ...

    // 方法
    [修饰符] 返回值 方法名1([参数列表]) {
        方法体语句;
    }
    [修饰符] 返回值 方法名2([参数列表]) {
        方法体语句;
    }
    ...

    // 构造器：创建对象用的方法
    [修饰符] 类名([参数列表1]) {
        方法语句;
    }
    [修饰符] 类名([参数列表2]) {
        方法语句;
    }

}
```

类的三大部件：成员变量、方法、构造器。
